## 개요

이 저장소는 Windows 시스템에서 생산성을 향상시키기 위해 설계된 AutoHotkey 스크립트를 포함합니다. 주요 초점은 OpenAI의 ChatGPT API와 통합되는 AI 기반 질문 및 답변 도구입니다.

## 요구사항

- **AutoHotkey v2.0** 이상
- **Windows OS**
- **OpenAI API 키** (ChatGPT 스크립트용)
- **인터넷 연결** (API 요청용)

## 설치

1. [AutoHotkey v2.0](https://www.autohotkey.com/) 다운로드 및 설치
2. 이 저장소를 클론하거나 다운로드
3. OpenAI API 키 설정 (자세한 내용은 [설정](#설정) 참조)

## 스크립트

### ChatGPT 질문답변 프로그램 v1.0

**파일:** `AI질문답변.ahk`

ChatGPT API와 상호작용하기 위한 간단한 GUI 애플리케이션입니다.

#### 기능:
- 기본 질문 및 답변 인터페이스
- GPT-4o 모델 지원
- 간략한 답변 옵션 체크박스
- HTML 형식 응답 옵션
- ChatGPT 웹사이트 직접 링크
- 답변 복사 기능

#### 사용법:
1. `OPENAI_API_KEY` 환경 변수 설정
2. 스크립트 실행
3. 입력 필드에 질문 입력
4. "질문하기" 버튼 클릭
5. "답변 복사" 버튼을 사용하여 답변 복사

### ChatGPT 질문답변 프로그램 v2.0

**파일:** `AI질문답변2.ahk`

대화 기록, 다중 AI 서비스 지원 및 개선된 UX를 갖춘 향상된 버전입니다.

#### 기능:
- **자동 선택 텍스트 캡처**: 프로그램 시작 시 선택된 텍스트를 자동으로 복사
- **대화 기록**: 여러 질문에 걸쳐 컨텍스트 유지
- **다중 AI 서비스 통합**: 
  - ChatGPT
  - Claude (Anthropic)
  - Perplexity
- **키보드 단축키**:
  - `Enter`: 질문 제출
  - `Shift + Enter`: 입력 필드에서 새 줄
  - `Escape`: 프로그램 종료
- **대기 메시지 대화상자**: API 요청 중 진행 상황 표시
- **답변 길이 제한**: 1000자 이상의 응답 자동 잘림
- **향상된 UI**: 색상 코딩된 버튼 및 개선된 레이아웃
- **AI 서비스로 전송**: 질문이나 답변을 웹 기반 AI 서비스로 직접 전송

#### 사용법:
1. 모든 애플리케이션에서 텍스트 선택 (선택사항)
2. 스크립트 실행 - 선택된 텍스트가 자동으로 채워짐
3. 질문 입력 또는 수정
4. `Enter` 키를 누르거나 "질문하기" 버튼 클릭
5. 후속 질문을 위해 대화 기록 사용
6. 전용 버튼을 사용하여 다른 AI 서비스로 질문/답변 전송

### 자동화 핫키 스크립트

**파일:** `오토핫키모음.ahk`

다양한 Windows 자동화 기능을 제공하는 종합 스크립트입니다.

#### 주요 기능:

**1. 주소 자동 입력 (핫스트링)**
- `addr1`: 경기도 하남시 미사강변대로 95, 미사강변센트럴자이 102동102호
- `addr2`: 서울시 동대문구 전농로 22, 엠제이빌딩 5층 (주)썸유
- `addr3`: 경기도 포천시 소홀읍 호국로 302-20 5동
- `addr4`: 경기도 하남시 조정대로 45, F8층 821호(풍산동,미사센텀비즈)
- `tod`: 오늘 날짜 자동 입력 (yyyy-MM-dd 형식)

**2. 프로그램 실행 단축키**
- `Ctrl + Alt + N`: 메모장 실행
- `Ctrl + Alt + E`: 엑셀 실행
- `Ctrl + Alt + C`: Windows 계산기 실행
- `Ctrl + Alt + P`: 파워포인트 실행
- `Ctrl + Alt + J`: Jupyter Notebook 실행 (Anaconda 환경)
- `Win + 1`: Notion 실행
- `Win + 2`: Cursor AI 실행
- `Win + C`: AI질문답변2 실행
- `Win + I`: 거래명세서확인 실행
- `Win + M`: 상품소싱 메모장 템플릿 생성
- `Win + 4`: Figma 실행

**3. 웹사이트 바로가기**
- `Ctrl + Alt + Y`: YouTube 열기
- `Ctrl + Alt + G`: ChatGPT 열기
- `Ctrl + Alt + S`: 쿠팡서플라이어 허브 로그인
- `Ctrl + Alt + W`: 쿠팡윙 로그인
- `Ctrl + Alt + M`: 메이크샵 로그인
- `Ctrl + Alt + V`: 네이버 스마트스토어 로그인
- `Win + O`: 올라 로그인
- `Win + T`: EcountERP 로그인

**4. 유틸리티 단축키**
- `F8`: 마우스 자동 클릭 (키를 누르고 있는 동안 지속 클릭)
- `Ctrl + Alt + X`: 현재 활성화된 창 닫기

#### 사용법:
1. 스크립트를 실행하면 백그라운드에서 실행됩니다
2. 각 기능에 해당하는 단축키를 누르면 자동으로 실행됩니다
3. 핫스트링 기능은 텍스트 입력 시 자동으로 변환됩니다 (예: `addr1` 입력 후 스페이스바)

#### 설정:
- **마우스 클릭 간격**: `ClickInterval` 변수로 조절 가능 (기본값: 100ms)
- **기본 브라우저**: `DefaultBrowser` 변수로 변경 가능 (기본값: chrome.exe)
- **파일 경로**: 일부 기능은 파일 경로가 하드코딩되어 있으므로 사용자 환경에 맞게 수정 필요

### 오늘 날짜 자동 입력

**파일:** `오늘날짜.ahk`

간단한 날짜 자동 입력 스크립트입니다.

#### 기능:
- `today` 입력 시 현재 날짜를 `yyyy-MM-dd` 형식으로 자동 입력

#### 사용법:
1. 스크립트 실행
2. 텍스트 입력 필드에서 `today` 입력 후 스페이스바 또는 엔터
3. 자동으로 오늘 날짜가 입력됨 (예: 2025-04-28)

## 설정

### OpenAI API 키 설정

1. **API 키 가져오기**: [OpenAI Platform](https://platform.openai.com/account/api-keys)에서 API 키 획득

2. **환경 변수 설정**:
   
   **Windows (PowerShell):**
   ```powershell
   [System.Environment]::SetEnvironmentVariable("OPENAI_API_KEY", "your-api-key-here", "User")
   ```
   
   **Windows (명령 프롬프트):**
   ```cmd
   setx OPENAI_API_KEY "your-api-key-here"
   ```
   
   **Windows GUI:**
   - 시스템 속성 → 환경 변수 열기
   - 새 사용자 변수 추가:
     - 변수 이름: `OPENAI_API_KEY`
     - 변수 값: API 키

3. 변경 사항을 적용하려면 **터미널/애플리케이션 재시작**

### API 모델 설정

- **v1.0**: `gpt-4o` 모델 사용, `max_tokens: 500`
- **v2.0**: `gpt-4o-mini` 모델 사용 (토큰 제한 미지정)

모델을 변경하려면 스크립트를 편집하세요:
- v1.0: 94번 줄 - `model` 및 `max_tokens` 값 수정
- v2.0: 210번 줄 - `requestObj`의 `model` 값 수정

### 자동화 핫키 스크립트 사용법

**오토핫키모음.ahk**는 백그라운드에서 실행되며, 다양한 단축키를 통해 기능을 실행합니다.

#### 주요 사용 시나리오:

1. **주소 입력**: 문서 작성 시 `addr1`을 입력하고 스페이스바를 누르면 자동으로 주소가 입력됩니다
2. **날짜 입력**: `tod`를 입력하면 오늘 날짜가 자동으로 입력됩니다
3. **빠른 프로그램 실행**: `Ctrl + Alt + E`로 엑셀을 즉시 실행할 수 있습니다
4. **웹사이트 빠른 접근**: `Ctrl + Alt + G`로 ChatGPT를 바로 열 수 있습니다
5. **마우스 자동 클릭**: 게임이나 반복 작업 시 `F8` 키를 누르고 있으면 자동으로 클릭됩니다

#### 주의사항:
- 일부 기능은 특정 프로그램이 설치되어 있어야 작동합니다 (예: Notion, Cursor, Figma)
- 파일 경로가 하드코딩된 기능은 사용자 환경에 맞게 수정이 필요합니다
- 마우스 자동 클릭 기능은 게임이나 특정 애플리케이션에서 사용 시 주의가 필요합니다

## 문제 해결

### API 키를 찾을 수 없음

**오류:** "환경 변수 'OPENAI_API_KEY'가 설정되지 않았습니다"

**해결 방법:**
1. 환경 변수가 설정되었는지 확인: `echo %OPENAI_API_KEY%` (CMD) 또는 `$env:OPENAI_API_KEY` (PowerShell)
2. 변수 이름이 정확히 `OPENAI_API_KEY`인지 확인 (대소문자 구분)
3. 변수 설정 후 스크립트 재시작

### API 요청 오류

**오류:** "API 오류가 발생했습니다"

**가능한 원인:**
- 잘못된 API 키
- API 크레딧 부족
- 네트워크 연결 문제
- API 서비스 중단

**해결 방법:**
- API 키가 유효하고 크레딧이 있는지 확인
- 인터넷 연결 확인
- 서비스 상태 확인: [OpenAI Status](https://status.openai.com/)

### 텍스트 선택이 작동하지 않음 (v2.0)

**문제:** 선택된 텍스트가 자동으로 채워지지 않음

**해결 방법:**
- 스크립트 실행 전에 텍스트가 선택되어 있는지 확인
- 일부 애플리케이션은 Ctrl+C를 통한 클립보드 복사를 지원하지 않을 수 있음
- 질문 필드에 텍스트를 수동으로 붙여넣기

### 응답 잘림 (v2.0)

**문제:** 답변이 1000자에서 잘림

**해결 방법:**
- UI 오버플로우를 방지하기 위한 의도된 동작입니다
- 전체 응답을 보려면 "답변 복사"를 사용하여 텍스트 편집기에 붙여넣기
- 또는 전체 컨텍스트를 위해 웹 기반 AI 서비스로 전송
